What is UX?

"User experience" encompasses all aspects of the end-user's interaction with the company, its services, and its products.

Nielson Norman Group

UX vs UI

User interface design involves creating a user-friendly interface thats easy to use, elegant, and in working condition.

UX vs Usability

Usability is a quality attribute that assesses how easy user interfaces are to use.

Importance of UX

  • User centered design
  • Accessibility

Evaluation

A/B Testing

A study to compare the effectiveness and quality of experience of different user interfaces.

Wireframes and Prototypes

User Flows
